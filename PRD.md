# SEOCopy — AI SEO & Copy Generator

## Product
A web app that generates SEO meta tags and landing page copy using AI. Users input a URL or product description and get optimized SEO titles, meta descriptions, Open Graph tags, and landing page copy variants.

## Stack
- **Framework:** Next.js 16 (App Router, TypeScript, Tailwind CSS)
- **AI:** Anthropic Claude API (ANTHROPIC_API_KEY from ~/.config/openclaw/secrets.env)
- **Payments:** Stripe Checkout (STRIPE_SECRET_KEY from ~/.config/openclaw/secrets.env)
- **Deployment:** Vercel

## Pricing
- Free: 2 generations (no account needed, stored in localStorage)
- Pay-per-use: $5/generation via Stripe Checkout
- Subscription: $19/month unlimited via Stripe Checkout

## MVP Pages & Routes

### `/` — Landing Page
- Hero: "AI-Powered SEO Copy in Seconds"
- Value props: Save hours, conversion-optimized, SEO best practices
- CTA: "Generate Free" → scrolls to generator
- Pricing section (free trial, per-use, monthly)
- Footer with links

### `/generate` — Generator (core feature)
- Input form:
  - URL field (optional — will fetch and analyze the page)
  - OR Product/business description textarea
  - Industry/niche dropdown (optional)
  - Tone selector: Professional / Casual / Bold / Luxury
- Output (generated by AI):
  - SEO Title (60 char optimized)
  - Meta Description (155 char optimized)
  - OG Title + OG Description
  - 3 Landing Page Headline variants
  - 3 Landing Page Subheadline variants
  - CTA button text suggestions
- Copy-to-clipboard buttons on each output
- "Generate Another" button (decrements free counter or checks payment)

### `/api/generate` — API Route
- POST endpoint
- Accepts: { url?, description?, industry?, tone? }
- Checks free tier (via session/cookie) or payment status
- Calls Anthropic Claude API with structured prompt
- Returns generated copy as JSON
- Rate limited: 10 req/min per IP

### `/api/checkout` — Stripe Checkout Session
- POST endpoint
- Creates Stripe Checkout session for one-time ($5) or subscription ($19/mo)
- Redirects to Stripe hosted checkout
- Success URL: `/generate?paid=true`

### `/api/webhook` — Stripe Webhook
- POST endpoint
- Handles checkout.session.completed
- Stores payment status (cookie/session for one-time, or email lookup for subscription)

## Design
- Clean, modern, fast
- Tailwind CSS utility classes
- Dark/light mode support
- Mobile responsive
- Minimal — no unnecessary UI chrome

## Environment Variables Needed
```
ANTHROPIC_API_KEY=... (from ~/.config/openclaw/secrets.env)
STRIPE_SECRET_KEY=... (from ~/.config/openclaw/secrets.env)
STRIPE_WEBHOOK_SECRET=... (create after Stripe setup)
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=... (from Stripe dashboard)
```

## Implementation Order
1. Landing page (`/`) with Tailwind
2. Generator UI (`/generate`) — form + output display
3. AI generation API (`/api/generate`) — Anthropic integration
4. Free tier tracking (localStorage counter)
5. Stripe checkout integration (`/api/checkout`)
6. Stripe webhook handler (`/api/webhook`)
7. Connect payment status to generator access
8. Deploy to Vercel
9. Test end-to-end payment flow

## Do NOT
- Add authentication/signup for MVP (use cookies/localStorage)
- Over-engineer the database (no DB for MVP — cookie + Stripe is enough)
- Add admin panel
- Add user accounts
- Use any UI component library — plain Tailwind only
